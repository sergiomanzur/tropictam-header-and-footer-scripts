Easily add custom scripts to the header and footer of your theme directly from the WordPress dashboard.

== Description ==

TropicTam Header and Footer Scripts plugin provides a simple and intuitive settings page in the WordPress dashboard where users can add custom scripts to the header and footer of their theme. This is especially useful for adding tracking codes, custom CSS, or JS functionalities without having to edit theme files.

== Frequently Asked Questions ==

= How do I add scripts to the header? =

Navigate to **Tools** > **TropicTam Header and Footer** in your WordPress dashboard. In the **Header Code** textarea, add any scripts that you want to appear in the header of your site. Click **Save Changes**.

= How do I add scripts to the footer? =

Navigate to **Tools** > **TropicTam Header and Footer** in your WordPress dashboard. In the **Footer Code** textarea, add any scripts that you want to appear in the footer of your site. Click **Save Changes**.
